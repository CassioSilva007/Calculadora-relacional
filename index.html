Operadores Relacionais

Operador Símbolo Descrição Exemplo Resultado (se x=5, y=3)
Igual a == Verifica se os valores são iguais x == y false
Diferente de != Verifica se os valores são diferentes x != y true
Maior que > Verifica se o valor é maior que outro x > y true
Menor que < Verifica se o valor é menor que outro x < y false
Maior ou igual a >= Verifica se o valor é maior ou igual a outro x >= y true
Menor ou igual a <= Verifica se o valor é menor ou igual a outro x <= y false
